If you experience any problems with your Robox printer or have questions about operating it visit our support site. We're here to help!

You can view this README anytime from within AutoMaker by navigating to the About page (press the question mark icon) and then click on 'View README'.

Version 2.00.02
This is a bug-fix release.

Key changes:

  • Increased number of solid top and bottom layers to 10 in default fine profiles.
  • Increased number of solid bottom layers to 3 in default draft profiles.
  • Fixed fill density slider so that dragging to 0% and 100% correctly adjust the printed parts.
  • Made fill density editable on settings page.
  • Improved firmware upgrade process and compatibility with old versions of firmware.
  • Improved display on small screens (e.g. 1366x768) - fixed issue where settings panel was not visible.
  • Manual Jog Arrows and Extrusion buttons now have correct cursor.
  • Y Motion Jog Arrows positions corrected.
  • It is possible to create projects when no printers are connected (again!).
  • Ungrouping of stl's on plating page does not allow parts to move under the bed.
  • Head Reset to Defaults now offers different head types.
  • Temperature graph on status side panel appears/disappears dependent on available space.
  • Update java to u74 - stability + videos in My Mini Factory cloud model repository now work.
  • Changing user-selected filament or head or reselecting a printer now forces time and cost recalculation.
  • Split win32 and win64 installers to reduce windows installer size.
  • Inhibited filament library / settings selection during calibration / purge.
  • Decimal Separator on Layout Page - resizing using dimension entry boxes now works when separator is a comma.
  • 'Eject Stuck Material 2' now inhibited when a single material head is installed.
  • User is informed that no model are on bed on the settings page and the make button is inhibited.
  • Settings panel stays on screen even when zoomed in close to model on layout page.
  • Filament library panel now allows spaces to be entered in name field.
  • Fixed support material setting logic - now preserves settings correctly on save.
  • Manual filament selection persists when switching between printers.
  • Slic3r no longer selectable as it has been deprecated.
  • Improvements around filament slip detection and the settings that control it.
  • Large filament slip thresholds now permitted.
  • Support for new head types.
  • Filament slip retries suppressed during pause.
  • Loading filament now moves the head to X190 to ensure smooth travel of the filament down the bowden tube. If the robox has not been homed, it homes X first.
  • Feed rate multiplier now selected according to the extruder which is active.
  • Feed rate multiplier only applied to moves which involve one or more of {X,Y,Z} and one or more of {E,D,B}.
  • Bug Fixed around Feed rate multiplier and head types.
  • Reprint (double-click eject button) now suppressed if smart reel(s) not present.
  • Altered purge required logic.
  • First layer nozzle is now correctly chosen from profile when using a single material head.
Version 2.00.01

This release introduces dual-material printing to Robox. By upgrading to dual-material hardware you can print in two colours or choose to print with two different materials.

We've changed AutoMaker to make it easier to create successful prints. More information about this release can be found here

  • Dual-material head upgrade
    • This version of AutoMaker supports the RBX01-SM single material head as well as the new RBX01-DM dual-material head. Using this head you can print in two colours or use one material as support for the print.
    • Supports .obj multi-material files
    • Import more than one stl at once to group aligned parts together.
    • Buy a Dual Material Kit now at the Robox shop in the UK or click here to find out how to buy locally.
  • Status Page
    • Printer status is now displayed on the left hand side of the screen at all times.
    • If you're not using Robox reels you can choose your filament as soon as the material is loaded.
    • The 'advanced' pull out tab has been removed. All of the functions that were in this tab can be found on the Preferences page - just click the gear icon at the top of the screen.
  • Layout Page
    • The Layout page no longer allows colours to be chosen - we felt this was confusing. Instead you will see blue objects where material 1 is chosen and orange objects where material 2 is chosen.
  • Settings Page
    • Filament and Print profiles can now be accessed from any screen using the library icon at the top of the screen.
    • Print settings (for example material choices) will be automatically altered based on the hardware configuration you have.
    • The Settings page shows the printed parts in their correct colour based on the material loaded or chosen.
    • Material choices are changed automatically on the Settings screen if they aren't supported by your current hardware configuration.
    • Try to print a dual colour model with a single material head and you'll see a single colour model on the settings page.
  • Support Material
    • If you're using a dual-material head you can choose which material you want to use to print raft, brim and support.
  • Print Management
    • We've introduced SurePrint - a behind the scenes print management system which detects and attempts to resolve problems then alerts the operator if more input is required.
    •  SurePrint works even when Robox isn't connected to AutoMaker.
  • Beta features - use with caution
    • Timelapse images
      • We've added GoPro WiFi support. Now you can make your own timelapse videos of the awesome things you print!

Lean/Twist/Turn
---------------
It is now possible to set the lean, twist and turn of an object on the bed. Currently this can only be done using
text fields but a future release will provide a graphical interface:
Turn   - when looking down on the object from above the bed, the turn rotates the object clockwise on the bed.
Lean   - this indicates how many degrees to lean the object away from the vertical. 90 degrees will lay the object
         on its side and 180 degrees will turn it upside down.
Twist  - when looking from the top of the object down its vertical axis, the twist rotates the object clockwise around
         that vertical axis.

The lean is applied first, then the twist and finally the turn.
